# Padrão Iterator - Basquete

Implementação do padrão de projeto Iterator aplicado ao contexto de estatísticas de times de basquete.

## Como Executar

### Pré-requisitos
- Java 11 ou superior
- Maven 3.6 ou superior

### Compilar o projeto
```bash
cd basquete
mvn clean compile
```

### Executar os testes
```bash
mvn test
```

### Limpar e executar tudo
```bash
mvn clean test
```
